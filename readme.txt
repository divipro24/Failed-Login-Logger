== Description ==

Плагин безопасности. Этот плагин записывает все неудачные попытки входа в админ панель.

Какие данные записываются:

* Время
* Username
* Password
* IP Address
* Плагин НЕ записывает удачные попытки входа.


Таким образом вы можете отследить проводят ли атаки на ваш сайт и вы сможете предпринять действия для защиты от них.
